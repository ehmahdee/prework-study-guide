HTML

  • HTML is the scaffolding of websites.
  • The head element contains information about the web page.
  • The body element represents the visible content shown to the user.
  • Meta contains info about the page thats used by the browser.
  • Child elements are elements nested inside of a body also known as Parent elements.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS controls the style and look of websites.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: Apply to a specific HTML tag. Uses a style attribute with a CSS rule to style a particular page element.
  • Inline is only useful for quick and permanent changes and they are less flexible than Internal Style Sheets.
  • External Style Sheets are the most common way to style a webpage.
  • ESS Allows developers to keep CSS rules in a separate file. Link by using < link > in the head.

Git

  • GitHub allows teams to work asynchronously on the same project without overwriting each others work. Neat!
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript builds the logic of websites. Basically makes it interactive and fuctional.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is an object-oriented programming language, or OOP.
  • "for" loops use the pattern of indices to perform a task.
  • "falsy" is 0, -0, on, " ", null, undefined, NaN, false (boolean)
  • "truthy" is all other values, including true (boolean)
  • JS indices always start at 0, not 1.
  • Indices correspond to arrays. if var letters = [ "A", "B", "C"] console.log(letters[0]) would show "A"
  • "for" loops have 3 statements. 1st statement determines the starting point
  • 2nd statement decides the condition. if number "x" is shorter than the length of the array, loop will keep running.
  • 3rd statement allows array to iterate over each item, Indices increase by 1. x++ is shortcut for x+1
  • Browser Application Programming Interfaces (APIs) can be built into web browsers.
